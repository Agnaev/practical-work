<h1>Практическая работа</h1>
<h2>Тема: Построение графика обработки n деталей универсальным обрабатывающим конвейером.</h2>
<p>На вход подается матрица времен переналадки станка для изготовления i-той детали. Результатом работы программы будет оптимальный порядок деталей, минимизирующий время обработки n деталей.</p>
<p>Поставленная задача схожа с задачей коммивояжера (<abbr title="Traveling Salesman Problem">TSP</abbr>). Поэтому выбран метод ветвей и границ (Branch and bound) для решения поставленной задачи.</p>
